---

## 📖 Overview

USA COVID-19 Live Map is an interactive web application that visualizes COVID-19 statistics across the United States. The platform presents state-wise data through an interactive map and dashboard, allowing users to monitor confirmed cases, recoveries, deaths, and other pandemic-related statistics in an intuitive way.

---

## ✨ Features

- 🗺 Interactive USA map
- 📊 State-wise COVID-19 statistics
- 🔄 Real-time data updates
- 📈 Interactive charts and visualization
- 📍 Clickable state information
- 📱 Responsive interface
- ⚡ Fast data loading
- 🎨 Clean dashboard design
